## Command Protocols

`/` "Slash Commands"を入力して、以下のプロトコルにアクセスしてください。

### Backup & Restore
- `/audit backup` : サーバー全体の高精度バックアップ。
- `/backup channels` : チャンネル構成の保存。
- `/backup roles` : ロール権限の完全記録。
- `/restore_channels` : チャンネル構造の強制復旧。
- `/restore_roles` : ロール権限の緊急ロールバック。

### Defense & Moderation
- `/safe ban` : 誤操作防止機能付き高精度BAN。
- `/safe_kick` : 迅速な排除プロトコル。
- `/deny` / `/allow` : 特定の権限を動的にロック/アンロック。
- `/set_approval` : 承認フローシステムの構築。

### System Utility
- `/stats` : サーバーの防衛統計を表示。
- `/status` : Botの稼働状況を確認。
- `/show_config` : 現在のセキュリティ設定の可視化。
- `/set_log` : ログ出力先チャンネルの指定。
